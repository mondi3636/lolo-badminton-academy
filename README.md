# Badminton Academy Manager

A simple web app to manage badminton academy operations including players, attendance, fees, and sessions.

## Features

- **Players Management**: Add, view, and manage player details
- **Attendance Tracking**: Mark daily attendance for players
- **Fees Management**: Track monthly fee payments with automatic calculations
- **Sessions & Coaches**: Schedule training sessions and assign coaches
